# Grok Trader Chrome Extension

AI-powered prediction market analysis for Polymarket.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build the extension:
   ```bash
   npm run build
   ```

3. Load the extension in Chrome:
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the `client` directory

## Development

Run the dev server:
```bash
npm run dev
```

After making changes, rebuild and reload the extension in Chrome.

## Features

- Collapsible sidebar on Polymarket event pages
- Professional UI matching Polymarket's design
- AI-powered market analysis (coming soon)
